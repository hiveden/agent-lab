这篇文章的核心观点是什么？
GPT-5 API 的核心突破是原生多模态推理——不再需要拼接多个模型来处理图文音视频。对 Agent 工程师来说，最大的变化是可以用单个 API 调用完成跨模态任务编排，pipeline 设计大幅简化。
对我们当前的 Radar 项目有什么影响？
直接影响不大，因为 Radar 主要处理文本内容。但如果后续扩展到视频/播客类信息源，GPT-5 的多模态能力可以让采集链路更简洁。
